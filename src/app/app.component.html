Statistical Metrics
When your model has been trained we use the equation of the best fit line to calculate some other statistical indicators for your training data set that tells you how good the line represents or fits your data set.
Regression is a statistical data analysis technique which is used to determine the extent to which there is a linear relationship between a dependent variable \(y \) and one or more independent variables \(x\)

In other words, given a number of \( (x,y) \) pairs, we can use regression to find the equation of a line $$y = mx + c $$ that best describes the \( (x,y) \) data by minimizing the sum of squared differences between the predicted target variable \(\tilde y\) and the actual target variable \(y\)
